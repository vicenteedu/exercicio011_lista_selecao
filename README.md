# Exercicio 11 da lista de seleção (Faculdade)

Este projeto tem como objetivo implementar, na linguagem Java, a resolução de uma equação do segundo grau no formato ax² + bx + c = 0.

O programa solicita ao usuário os coeficientes a, b e c, verifica se a equação é válida (ou seja, se a é diferente de zero) e calcula o valor do discriminante (Δ). A partir desse valor, o sistema determina a quantidade de raízes reais da equação:

Se Δ < 0 → não existem raízes reais
Se Δ = 0 → existe uma única raiz real
Se Δ > 0 → existem duas raízes reais

Os resultados são exibidos ao usuário por meio de janelas gráficas utilizando JOptionPane.

Desenvolvido utilizado JDK 21 e NetBeans 29
